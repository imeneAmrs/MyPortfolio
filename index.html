Web Scraper Tool

    Used Technologies :  
  • Java
  • JSoup Library
  • Natural Language Processing

In order to collect a meaningful data about a particular module, the idea of this tool was born, we did extract data automatically from the web by collecting a set of urls based on a number of google search mentioning some keywords or combination of keywords (requesting the web browser ), this desired list of urls will be downloaded, and since w're interested in text only , we'll parse these html pages by retrieving all text in between html tags, by this way we'll collect enough data all saved in text files.
The challenge faced in this app was when doing multiple requests in small amount of times, my IP address will be blocked for an undefined time, so a timeout between a number of requests did the pupose.
Beside this feature, this tool allows us to clean the gathered data, this includes corpus normalisation & stemming and so many other options.
The resulted corpus will be used for NLP tasks.
